Additional user data

*This data will be used to update the user if it already exists, or create a new one.

Concurrent Chat value should be between -1 and 20 (-1: no limit, 0: disabled). Default: disabled.
Concurrent Voice value should be between 0 and 1 (0: disabled, 1: enabled). Default: disabled.
Concurrent Video value should be between 0 and 1 (0: disabled, 1: enabled). Default: disabled.
Don't overwrite user's data already in Vivocha.
